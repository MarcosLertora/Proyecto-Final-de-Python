## DESCPRIPCIÓN GENERAL DEL SISTEMA
El Sistema de Turnos Hospitalarios es una aplicación desarrollada en Python que se ejecuta por consola y permite administrar la atención de pacientes en un centro de salud.
El sistema posibilita registrar pacientes, buscar pacientes por DNI, asignar turnos médicos, controlar la disponibilidad de horarios, atender y cancelar turnos, además de generar estadísticas sobre la atención realizada.
Durante el desarrollo se implementaron estructuras condicionales, estructuras repetitivas, funciones, validaciones de datos, acumuladores, contadores y modularización del código, organizando las distintas funcionalidades en módulos independientes para facilitar su mantenimiento y comprensión.

## FUNCIONALIDADES DEL SISTEMA
    - Registro de pacientes.
    - Búsqueda de pacientes por DNI.
    - Asignación de turnos médicos.
    - Validación de disponibilidad de horarios.
    - Atención de pacientes.
    - Cancelación de turnos.
    - Consulta de pacientes registrados.
    - Consulta de turnos registrados.
    - Generación de estadísticas de atención.
    - Validación de datos ingresados por el usuario.

## INSTRUCCIONES DE EJECUCIÓN
1. Descargar o clonar el repositorio.
2. Abrir la carpeta del proyecto en Visual Studio Code o cualquier otro editor compatible con Python.
3. Abrir una terminal dentro de la carpeta del proyecto.
4. Ejecutar el archivo principal del sistema con el siguiente comando: python main.py (o python3 main.py en algunos sistemas operativos).
5. Una vez iniciado el programa, seguir las opciones del menú que se muestran en pantalla para utilizar el sistema.
